Getting Started with Docker

Follow these steps to start using Docker:

1. Install Docker

Download and install Docker from the official site: Docker Official Website

2. Verify Installation

Run the following command in your terminal or command prompt:

docker --version

3. Run Your First Container

Use this command to run a simple Hello World container:

docker run hello-world

4. List Running Containers

Check running containers with:

docker ps

5. Stop a Running Container

Use the container ID to stop it:

docker stop <container_id>

6. Remove a Container

Remove an unused container with:

docker rm <container_id>

7. Pull an Image

Download an image from Docker Hub:

docker pull ubuntu

8. Run an Interactive Shell

Run an interactive terminal inside a container:

docker run -it ubuntu bash
